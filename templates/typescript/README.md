# TypeScript Template

TypeScript development shell template.

## Quick Start

```bash
nix flake init -t github:psoland/os-config#typescript
echo "use flake" > .envrc
direnv allow
```

## Included Tools

- `nodejs_20`
- `pnpm`
- `typescript`
- `typescript-language-server`
- `prettier`
- `eslint`
- `just`
